Willis College

moodle cms
CSS
web development

I was excited to work on this project seeing that I would be developing yet another educational tool to help this great institution empower its students with crucial knowledge as it's been doing for over 150 years.
I designed a CMS platform based on Moodle to mirror the school's courses' structure, prototyped and tested the platform on a LAMP VM prior to implementation. Also liaised regularly with the school's administration gathering in-depth requirements and project scope so to be sure the platform would be as efficient as possible.
It was a challenging and fun way to give back to an institution that has given me so much.

Magnifi

android
app design

Magnifi is a global mobile on-demand knowledge sharing marketplace, connecting experts with knowledge seekers through paid, real-time audio/video consultation.
In this project, the goal was to mitigate the confusion users had when dealing with Magnifi's Social Feeds Android app screen.
Their input quite often resulted in broken links on their profile page and the implemented solution was a feature enhancement of the input fields and a clear explanation on how to fill the social link fields.
